Sistema de analisis de estadistica criminal y visualizacion geografica

Tesis de grado

El proposito de mi proyecto de tesis es el de crear un sistema de soporte de decisiones que analice data estadistica criminal, aplicando metodos estadisticos y de mineria de datos para generar patrones descriptivos y/o predictivos de comportamiento criminal. Los patrones generados serian representados en servicios de mapas web para facilitar su compresión. Para este proyecto se utiliza python, spark SQL, Json, GeoJson y Mapbox como servicio de mapa web

La data de prueba utilizada para este proyecto proviene de la oficina de estadistica del Reino Unido, la cual es gratuita y de libre acceso

  • Periodo: 2016 - Presente
  • Cliente: UNET
  • Categoria: Ciencia de Datos
